## BOT COMMANDS.

Bot Prefix - ,, <br>

play <name/URL> - play music in a voice channel.  <br>
search <name> - open a panel to choose a music and then play it.<br>
pause - pause the current music.<br>
resume - puts the current music back on.<br>
queue - see the next songs.<br>
clear-queue - remove music in the queue.<br>
shuffle - to mix the queue.<br>
nowplaying - see music in progress.<br>
loop - to enable or disable the repeat function.<br>
volume <1 - 100> - change the volume.<br>
skip - skip to next music.<br>
stop - stop all music.<br>
filter <filter> - add / remove filter.<br>
w-filters - see filters.<br>
